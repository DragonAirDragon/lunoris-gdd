# Prompt System

> Interaction hints

---

## Description

UI element that appears on screen when approaching an interactive object. Shows which button to press and what can be interacted with.

---

## Elements

| Element | Description |
|---------|-------------|
| **Button** | Which key to press |
| **Name** | What we interact with |

---

## Where Used

All scenes:
- DayHome
- DreamHub
- NightmareHub
- Tutorial
- Elar
- Car Level
- Nightmare Level
