## 📂 Структура

```
├── index.md              ← Главная страница
├── Player/               ← Управление, камера, статы
├── Entities/             ← Собака, враги
├── Runner/               ← Гоночный режим
├── World/                ← Мир, уровни
├── Progression/          ← Сохранения, прогресс
├── UI/                   ← Интерфейс
├── Audio/                ← Звук
├── Narrative/            ← Диалоги
├── Core/                 ← Базовые системы
├── Effects/              ← VFX, постобработка
├── Templates/            ← Шаблоны страниц
└── images/               ← Скриншоты, гифки
```

## 🚀 Локальная работа

1. Клонируй репозиторий
2. Редактируй `.md` файлы в VS Code
3. Push → GitBook обновится автоматически

## 🖼️ Добавление картинок

1. Положи файл в `images/`
2. Вставь в Markdown: `![Описание](images/filename.png)`

---

## 📝 Как добавить страницу

1. Скопируй шаблон из `Templates/`
2. Заполни секции
3. Добавь ссылку в `index.md`
